# 数独

ブラウザで動く数独パズルゲームです。

## 技術スタック

- React + TypeScript + Vite

## 起動方法

```bash
npm install
npm run dev
```

ブラウザが自動で開きます。開発サーバーはアプリ内の「終了」ボタンで停止できます。

## 機能

### ゲームプレイ
- 難易度は初級・中級・上級・エキスパートの4段階
- タイマー表示
- セルを選択して数字を入力（マウスまたはキーボード）

### 操作
| 操作 | 動作 |
|------|------|
| `1`〜`9` | 数字入力 |
| `Backspace` / `Delete` / `0` | 消去 |
| `N` / `スペース` | メモモード切替 |
| `↑↓←→` | セル移動 |
| `Ctrl+Z` | 1手戻す |

ボタン上にも主要なショートカット（`Ctrl+Z` / `N` / `Del`）が表示されています。

### メモ機能
メモモードをオンにすると、セルに候補数字を複数書き込めます。選択中のセルと同じ数字が確定しているマスでは、その数字が青い丸バッジで強調表示されます。メモ内の同じ数字も丸バッジで強調されます。

### ヒント機能
- **答えを見る**: 選択中のセル（未選択なら最初の未確定セル）に正解を表示
- **次の手を教える**: ソルバーが次に置けるマスを技法名とともに提示。使用した技法に応じて以下のメッセージが表示されます
  - 一択確定（Naked/Hidden Single） — 候補が1つに絞れるマス
  - ペア・三つ組消去（Locked Candidates/Pair/Triple） — 中級消去後に確定するマス
  - X-Wing・XY-Wing・Swordfish消去 — 上級消去後に確定するマス
  - エキスパートレベルで手がかりがない場合は「答えを見る」を使用してください

### 1手戻す
「戻す」ボタンまたは `Ctrl+Z` で直前の操作を1手だけ取り消せます。

## 難易度の基準

技法ベースのソルバーで難易度を自動分類しています。

| 難易度 | 必要な技法 |
|--------|-----------|
| 初級 | Naked Single / Hidden Single のみ |
| 中級 | Locked Candidates、Naked/Hidden Pair・Triple |
| 上級 | X-Wing、Swordfish、XY-Wing、XYZ-Wing |
| エキスパート | 上記技法では解けないパズル |
